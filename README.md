# عسير - التجربة الذكية
## Asir Smart Experience – منصة عسير الذكية للتجارب السياحية

منصة رقمية ذكية لتحويل السياحة في عسير من موسمية إلى تجربة مستدامة طوال العام عبر:
- التخصيص الذكي
- المواسم السياحية
- دعم الاقتصاد المحلي
- التفاعل الرقمي
- توحيد الخدمات السياحية

### المحور المركزي – جواز عسير (Asir Passport)

يتم تنفيذ نظام جواز عسير محلياً باستخدام SQLite ويشمل:
- أختام رقمية للأماكن
- نقاط
- أوسمة
- تحديات
- مكافآت
- خصومات (وهمية – عرض فقط)
- شهادة سياحية رقمية
- سجل الرحلات
- أختام موسمية: الضباب، البن، الصيف، الشتاء

### الأقسام الأساسية

| القسم | الوصف |
|-------|-------|
| وش فيه اليوم؟ | فعاليات اليوم، نهاية الأسبوع، أنشطة عائلية، حفلات |
| جوّك اليوم | حالة الطقس، اقتراحات، تنبيهات الضباب والمطر |
| نخيم؟ | مواقع التخييم، تنبيهات أمان، تجارب ليلية |
| نطلع نتمشى | مسارات طبيعية، شلالات، غابات، نقاط تصوير |
| قهوتنا غير | مزارع البن، المقاهي، تجارب القهوة، خصومات جواز عسير |
| من زمان | قرى تراثية، متاحف، قصص نصية، ختم "عشت التراث" |
| من أهلها | أسر منتجة، متاجر محلية، أكشاك موسمية |
| مواسم عسير | الضباب، الصيف، الشتاء، البن، التخييم، الفعاليات |
| الخدمات السياحية | السكن، المواصلات، ميزة الحجز الذكي |
| التجارب الغذائية | مطاعم، فود ترك، توصيات |
| التسوق والترفيه | أسواق، مولات، حرف يدوية |
| السياحة الساحلية | القحمة، البرك، الشواطئ، الصيد |

### التقنيات المستخدمة

- Flutter 3.32.5
- Dart
- SQLite (محلي – sqflite)
- Python + Flask (Backend للتوصيات الذكية)
- نموذج توصيات ذكي (recommendation_engine.py)

### مكان الملفات

- **الشعار:** `assets/images/Logo.jpg`
- **الخط:** `assets/fonts/IBMPlexSansArabic-*.ttf` (Regular, Light, Medium, SemiBold, Bold, Thin, ExtraLight)

### خطوات التشغيل على Windows + VSCode

1. تأكد من تثبيت Flutter (إصدار 3.32.5 أو متوافق)
2. افتح المجلد في VSCode:
   ```
   D:\VSCode\Projects\flutterprojects\asir_smart_experience
   ```
3. نفّذ في الطرفية:
   ```
   flutter pub get
   ```
4. للتشغيل على المحاكي أو الجهاز:
   ```
   flutter run
   ```
5. لبناء APK (أندرويد):
   ```
   flutter build apk
   ```

### تشغيل الخادم الذكي (Flask + Python)

```powershell
cd D:\VSCode\Projects\flutterprojects\asir_smart_experience\backend
pip install -r requirements.txt
python app.py
```

يعمل الخادم على `http://localhost:5000`

### إعداد عنوان الـ API في التطبيق

- **المحاكي الأندرويد:** الافتراضي `10.0.2.2:5000` (يتوافق تلقائياً)
- **جهاز حقيقي:** غيّر `lib/core/api_config.dart` إلى IP الكمبيوتر، مثال: `http://192.168.1.10:5000`

### مثال تشغيل المشروع

**1. تشغيل الخادم أولاً:**
