# عسير — التجربة الذكية (Asir Smart Experience)

## 1. ملخص لغير التقنيين

**باختصار:** التطبيق يعرض لك عسير كوجهة سياحية: صور، أقسام (طقس، تخييم، قهوة، تراث…)، ويمكنك تسجيل الدخول والاستفادة من **جواز عسير** (نقاط، أختام، رحلات محفوظة على جهازك على الجوال، أو في الذاكرة مؤقتاً على المتصفح).

**الذكاء الإضافي:** عندما يعمل خادم Python، يقدّم اقتراحات مثل «وش فيه اليوم؟» أو اقتراحات حسب الطقس أو عند الحجز (سكن ومواصلات) — هذه تُجلب من الإنترنت من عنوان واحد عند رفع المشروع على Render.

**الصور من الإنترنت:** المواقع الأخرى أحياناً لا تسمح للمتصفح بعرض صورها مباشرة داخل تطبيق ويب؛ لذلك يمرّر التطبيق الصور عبر **خادمك أنت** (نفس عنوان الموقع) فيظهر كل شيء بشكل طبيعي.

---

## 2. الفكرة والهدف

- جمع تجارب سياحية متنوعة في واجهة واحدة بالعربية (مع دعم إنجليزي للواجهة حيث يتوفر).
- تشجيع الاستكشاف عبر **جواز عسير**: زيارة أماكن، جمع أختام، نقاط، وسجل رحلات.
- دعم **توصيات ذكية** من الخادم عند توفر الاتصال بـ API.
- إتاحة **الويب** و**الأندرويد** من نفس المشروع البرمجي.

## 4. أقسام التطبيق مع أمثلة واقعية

| القسم | مثال واقعي لاستخدامه |
|--------|------------------------|
| **الرئيسية + Hero** | صورة بانوراما لعسير ثم قائمة أقسام مصورة |
| **جواز عسير** | تسجيل دخول ثم عرض النقاط، الأختام، الرحلات، الحجوزات |
| **دليل الأماكن** | تصفّح أماكن مقترحة مع صور ووصف |
| **وش فيه اليوم؟** | فعاليات واقتراحات؛ إن وُجد الخادم تُعرض توصيات «ذكية» |
| **جوّك اليوم** | طقس واقتراحات أنشطة مناسبة |
| **خرائط** | مدخل لاستكشاف المنطقة |
| **نخيم؟** | أفكار تخييم وصور |
| **نطلع نتمشى** | مسارات وطبيعة |
| **قهوتنا غير** | مزارع ومقاهي وبن عسيري |
| **من زمان** | تراث وقرى |
| **من أهلها** | منتجات محلية وأسر منتجة |
| **مواسم عسير** | ضباب، صيف، شتاء، إلخ |
| **الخدمات السياحية** | سكن ومواصلات؛ يرتبط باقتراحات الحجز من API |

## 5. كيف يعمل التطبيق وترابط الأجزاء

```
[شاشة البداية] → تهيئة المصادقة + قاعدة البيانات
       ↓
[تسجيل الدخول أو الرئيسية]
       ↓
[الأقسام] ← صور من AppImages (على الويب عبر بروكسي الخادم)
       ↓
[جواز عسير / الحجوزات / التفاصيل] ← DatabaseHelper (SQLite على الجوال، ذاكرة على الويب)
       ↓
[التوصيات الذكية] ← ApiService → Flask `/api/recommend/*`
```

- **المصادقة:** `AuthService` + `SharedPreferences` لحفظ معرف المستخدم بعد تسجيل الدخول.
- **البيانات المحلية:** `DatabaseHelper` — منفصل للمنصات (ملف `database_helper_io.dart` ونسخة ويب `database_helper_web.dart`).
- **الصور البعيدة على الويب:** `proxiedImageUrl` في `lib/core/image_url.dart` يحوّل الرابط إلى `…/api/remote-image?u=…` حتى يخدم Flask الصورة بنفس النطاق.
- **الخادم:** `backend/app.py` يقدّم API وملفات `build/web` بعد النسخ إلى `backend/web`.

## 6. المشاكل التي ظهرت والحلول

| المشكلة | السبب | الحل المعتمد |
|---------|--------|----------------|
| التطبيق يتعطل على الويب عند الانيميشن | `sqflite` ومسار الملفات لا يعملان في المتصفح | نسخة ويب من قاعدة البيانات في الذاكرة (`database_helper_web.dart`) |
| صور خارجية لا تظهر على الويب (CORS) | الخوادم الأخرى لا ترسل ترويسة السماح للمتصفح | بروكسي `/api/remote-image` على Flask؛ وفي Flutter تُمرَّر **كل** روابط الشبكة عبر `proxiedImageUrl` داخل `ImageTile` و`ImageListItem` وشاشة التفاصيل ورؤوس الأقسام (`proxiedImageUrl(AppImages.…)`) حتى تُحمَّل روابط `files.manuscdn.com` من البيانات دون خطأ |
| `gunicorn` غير موجود على Render | تثبيت الحزم أو مسار التشغيل | `start.sh` أو `python -m pip install …` ثم gunicorn |
| Service Worker بطيء أو أخطاء | بيئة استضافة باردة أو مهلة زمنية | `flutter build web --pwa-strategy=none` |
| مجلد `backend/web` فارغ على السيرفر | لم يُنسخ `build/web` | نسخ ناتج Flutter إلى `backend/web` ورفعه |

## 7. البنية التقنية ومقتطفات من الكود

**الواجهة:** Flutter (Dart)، إدارة حالة بـ Provider للغة، انتقالات مخصصة.

**عنوان الـ API (قابل للتغيير عند البناء):**

```dart
// lib/core/api_config.dart
static const String baseUrl = String.fromEnvironment(
  'API_BASE_URL',
  defaultValue: 'http://10.0.2.2:5000',
);
```

**تمرير الصور عبر الخادم على الويب فقط:**

```dart
// lib/core/image_url.dart
String proxiedImageUrl(String url) {
  if (!kIsWeb || url.isEmpty) return url;
  final base = ApiConfig.baseUrl.replaceAll(RegExp(r'/$'), '');
  return '$base/api/remote-image?u=${Uri.encodeComponent(url)}';
}
```

**استيراد شرطي لقاعدة البيانات:**

```dart
// lib/data/database_helper.dart
export 'database_helper_io.dart' if (dart.library.html) 'database_helper_web.dart';
```

**نقطة بروكسي الصور في Flask (مبسّط):** يُسمح فقط بنطاقات مُعرّفة في `_ALLOWED_IMAGE_HOSTS` لتقليل سوء الاستخدام.

## 9. التشغيل المحلي والنشر

**تشغيل Flask محلياً:**

```powershell
cd backend
pip install -r requirements.txt
python app.py
```

**تشغيل Flutter (موبايل):**

```text
flutter pub get
flutter run
```

**بناء الويب للنشر على Render (نفس عنوان الخادم):**

```text
flutter build web --release --dart-define=API_BASE_URL=https://اسم-خدمتك.onrender.com --pwa-strategy=none
```

ثم انسخ محتوى `build/web` إلى `backend/web` وادفع إلى GitHub. تفاصيل إضافية في `backend/DEPLOY_RENDER.md`.
